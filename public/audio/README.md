# Zen Mode — Ambient Audio Files

Place the following MP3 files in this directory for Zen Mode ambient audio:

- **rain.mp3** — Rain / thunderstorm ambient loop
- **white-noise.mp3** — White noise ambient loop
- **deep-focus.mp3** — Lo-fi / deep focus ambient loop

## Requirements

- Format: MP3
- Duration: At least 30 seconds (tracks loop automatically)
- Style: Ambient, calming, suitable for background listening while writing

## Suggested Sources (royalty-free)

- [Freesound.org](https://freesound.org) — Search for "rain loop", "white noise", "ambient focus"
- [Pixabay Music](https://pixabay.com/music/) — Free ambient tracks
- [Mixkit](https://mixkit.co/free-sound-effects/) — Free sound effects and loops
